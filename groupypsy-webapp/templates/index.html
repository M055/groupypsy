Assign Students to Projects

Instructions

Please upload the following four files to process your data:

  • Students File: Contains details of all students.
  • Projects File: Lists all available projects.
  • Requirements File: Specifies project requirements.
  • Choices File: Contains student project preferences.

Once all files are uploaded, click the Process button to generate the output file. Once you are done, please delete the data from the server by clicking the Clear Data button.

Please refer to the Help page for details of usage and of the files to be uploaded.
